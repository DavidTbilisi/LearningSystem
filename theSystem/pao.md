# PAO System — 10×10×10 (3-digit encoding)

Encodes **3-digit numbers** as Person + Action + Object using single digits.
Split: hundreds digit = Person, tens digit = Action, units digit = Object.

All items phonetically align to Major System consonants.

Example: `598` → Loki (5) + Burning (9) + Flag (8) = *Loki burning a flag*

## Worked examples

**`123`** → Thor (1) + Nailing (2) + Mask (3) = *Thor nailing a mask to the wall*

**`456`** → Robin Hood (4) + Launching (5) + Shield (6) = *Robin Hood launching a shield like a frisbee*

**`598 444 444`** (Georgian phone number):
- `598` → Loki (5) + Burning (9) + Flag (8) = *Loki burning a flag*
- `444` → Robin Hood (4) + Ripping (4) + Ring (4) = *Robin Hood ripping a ring*
- `444` → same scene repeated — impossible to forget
